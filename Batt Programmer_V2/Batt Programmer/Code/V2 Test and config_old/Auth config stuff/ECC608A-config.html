Instructions

The ATECC608A configuration explorer is broken up into three main sections:
Full Configuration
Full ATECC608A configuration (config zone) with the decoded meanings of each field.
Slot Configuration Summary
Summary of SlotConfig and KeyConfig values making it easier to see how all the slots are configured.
Configuration Import/Export
ATECC608A configurations are loaded, saved, and converted to different formats here.

Configuration Import/Export

Here is where an ATECC608A configurations are loaded into the utility and saved off again. The configuration data in the text box below represents the current config and updates in real-time to changes made above.
Import/Load Configuration
  1. Copy the configuration data from the text file it's saved in.
  2. Select the format the configuration data is in below.
  3. Paste the configuration data into the text box below.
Export/Save Configuration
  1. Select the format you want the exported configuration in.
  2. Copy the configuration data from the text box below.
  3. Paste the configuration data into a text file and save it.
